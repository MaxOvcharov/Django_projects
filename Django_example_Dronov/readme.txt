Важные замечания по поводу сайта:

 - Сайт полностью готов к работе.

 - Сайт не предназначен для публикации в Интернете (режим отладки включен, а списки администраторов и модераторов пусты, равно как и список допустимых доменов). На данный момент сайт предназначен лишь для изучения.

 - Имя пользователя для входа на сайт - admin, пароль - такой же. Данный пользователь имеет максимальные права (суперпользователь).

 - Через встроенный административный сайт доступны, в числе прочего, записи гостевой книги, позиции списка рассылки и хранилища изображений.

 - Список рассылки пуст.

 - Для всех страниц списка указан размер в две позиции. Это сделано для удобства отладки, чтобы можно было тут же проверить пагинацию.

 - Настройки почтового SMTP-сервера не указаны, и перед началом работы их следует задать.

 - Используются параметры журналирования по умолчанию.

 - Все файлы, не нужные для работы сайта, удалены.



Список файлов и папок, входящих в состав архива:

 - BROOMTRADE - папка проекта сайта.

 - !!!OTHERS - папка с дополнительными файлыми:

   - httpd.conf.add - пример конфигурации для Apache, настраивающий последний для публикации данного сайта (см. главу 32 книги). Предполагается, что папка сайта помещена в папку Sites, находящуюся в корне диска С; если это не так, следует соответственно изменить путь к ней. Данный код должен быть добавлен к содержимому файла httpd.conf.

   - settings.py.add - примеры конфигураций баз данных MySQL и PostgreSQL. В обоих случаях предполагается, что сервер установлен на том же компьютере, где развёрнут сайт, база данных имеет имя site, а подключение к ней выполняется через пользователя site с паролем site; если это не так, в код нужно вставить реальные данные о базе и подключении к ней. Данный код должен быть помещён в модуль settings пакета проекта.

   - 400.html, 403.html, 404.html и 500.html - файлы шаблонов страниц с сообщениями об ошибках.

 - readme.txt - пояснительный текст, обязательный для прочтения (этот файл).
